Introduction

Welcome to the AI-Driven and Spec-Driven Development educational book! This book is itself a demonstration of the methodologies it teaches, built using Spec-Kit Plus and AI-driven development practices.

What You'll Learn​

This book covers:

  1. AI-Driven Development: How artificial intelligence can accelerate software development while maintaining quality
  2. Spec-Driven Development: A methodology for building software through clear specifications, plans, and task breakdowns
  3. RAG Fundamentals: Understanding Retrieval-Augmented Generation and how it enables intelligent chatbots
  4. Implementation Guide: Step-by-step guide on how this project was built
  5. Chatbot Usage: How to use the embedded RAG chatbot to ask questions about this book
  6. Future Work: Ideas for extending and improving this project

About This Project​

This project is a hackathon demonstration that showcases:

  • Spec-Driven Development using Spec-Kit Plus framework
  • AI-Driven Development using Claude Code and AI assistants
  • RAG-based Chatbot using OpenAI, Qdrant Cloud, and FastAPI
  • Educational Book deployed to GitHub Pages using Docusaurus

Project Structure​

The project consists of:

  • Frontend: Docusaurus documentation site (this book)
  • Backend: FastAPI REST API with RAG capabilities
  • Vector Database: Qdrant Cloud for storing book embeddings
  • AI Services: OpenAI for embeddings and chat completions

How to Navigate​

Use the sidebar to navigate between chapters. Each chapter builds on previous concepts, so we recommend reading in order:

  1. Start with AI-Driven Development to understand how AI accelerates development
  2. Learn about Spec-Driven Development and the Spec-Kit Plus framework
  3. Explore RAG Fundamentals to understand how the chatbot works
  4. Follow the Implementation Guide to see how this project was built
  5. Learn How to Use the Chatbot to interact with the RAG system
  6. Check out Future Work for extension ideas

Interactive Features​

This book includes an embedded RAG chatbot that can answer questions about the content. You can:

  • Ask questions about any chapter using the chatbot interface
  • Select specific text and ask questions about only that selection
  • Get answers based strictly on the book content (no external knowledge)

Try it out in the Chatbot Usage chapter!
